  • 電子產品回收點地圖
  • demo | GitHub
  • 功能：
  • 使用者可查詢縣市地址，並且透過 Google Map 查找地點最近電子產品回收站；
  • 點擊回收站地標，可查詢地址以及查看接受回收的產品。
  • 使用技術：
  • Google Maps API：在 Google Maps 印出目標地點以及資料視窗
  • Geocoder：使用 Geocoder 將地址轉換成經緯度
  • AJAX：使用 AJAX 非同步傳遞技術載入資料
  • JSON & open-data：資料來源為環境保護署資料開放平台，資料來源格式為 JSON
  • BMI 計算器
  • demo | GitHub
  • 功能：
  • 使用者可輸入身高體重，計算出 BMI 值並且判斷為理想或是過重等不同等級；
  • 在計算當下並會記錄一切數據以及時間，方便使用者查看紀錄；
  • 使用者也可一使用狀況而消除個別紀錄。
  • 使用技術：
  • JavaScript ES6：使用 JavaScript 寫出計算邏輯
  • localStorage：使用 localStoage 將資料暫存於瀏覽器中
  • 高雄旅遊資訊
  • demo | GitHub
  • 功能：
  • 使用者可透過點擊熱門行政區按鈕或是選單查找目標地區；
  • 選擇地區後可看到地區景點開放時間、地點、電話及售票資訊。
  • 使用技術：
  • JavaScript ES6：使用 ES6 模版字符串（template literal）嵌入變數
  • AJAX：使用 AJAX 非同步傳遞技術載入資料
  • JSON & open-data：資料來源為高雄市政府資料開放平台，資料來源格式為 JSON
  • 前端工程：
  • JavaScript 基本語法、ES6
  • AJAX & XMLHttpRequest 非同步來源請求
  • API 串接 (Google Maps, FB Login)
  • 使用 Git GUI (Source Tree) 做版本控制
  • Web Layout：
  • 使用 HTML5、CSS3 開發靜態頁面
  • jQuery 互動效果
  • RWD 響應式網頁設計
  • Bootstrap 快速架構網頁框架
